║   🎯 TROUBLESHOOTING                                            ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝

❌ Email not sending?
   → Check .env has EMAIL_USER and EMAIL_PASSWORD
   → Verify App Password (not regular Gmail password)
   → Check 2FA is enabled
   → Look for errors in backend terminal

❌ "Cannot find module './utils/emailService'"?
   → File exists at: backend/utils/emailService.js
   → Try restarting backend: Ctrl+C then node server.js

❌ Database error "Unknown column 'is_verified'"?
   → Run migration: backend/migrations/add_otp_verification.sql
   → Verify: DESC users; (should show new columns)

❌ OTP says "Invalid OTP"?
   → Check if OTP expired (10 minutes validity)
   → Click "Resend OTP" and try new code
   → Verify email address is correct

❌ Login still blocked after verification?
   → Check database: SELECT is_verified FROM users WHERE username='email';
   → Should be 1 or TRUE
   → Try: UPDATE users SET is_verified = TRUE WHERE username='email';
